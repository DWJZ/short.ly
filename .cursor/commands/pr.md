# pr

create a new pull request on github. The commit message should follow the conventional commit message format like this:
- feat: xxx (add new feature)
- fix: xxx (fix a bug)
- docs: xxx (update documentation)
- refactor: xxx (refactor code)
- test: xxx (add tests)
- chore: xxx (other changes)
- style: xxx (style changes)
- perf: xxx (performance improvements)

Write your command content here.

This command will be available in chat with /pr
